# Fitness Center Yönetim ve Randevu Sistemi

Sakarya Üniversitesi Web Programlama dersi için geliştirilmiş ASP.NET Core MVC projesi.

## Özellikler
- Spor salonu / Hizmet / Eğitmen yönetimi (Yönetici paneli)
- Üyeler için randevu alma sistemi
- LINQ filtreleri ile REST API
- Yapay zeka tabanlı antrenman ve beslenme önerileri
